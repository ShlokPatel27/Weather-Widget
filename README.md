# Weather Widget

A beautiful and responsive weather application built with React, Vite, and Material UI. This widget provides real-time weather updates using a modern and clean user interface.

## Features

- Real-time weather data fetching using Axios
- Modern and responsive UI built with Material UI (MUI)
- Fast and optimized development experience with Vite
- Clean and intuitive design

## Technologies Used

- **React** (v18)
- **Vite**
- **Material UI** (MUI) for styling and icons
- **Axios** for API requests

## Installation & Setup

Follow these steps to run the project locally on your machine:

1. **Clone the repository** (if you haven't already):
   ```bash
   git clone https://github.com/ShlokPatel27/Weather-Widget.git
   ```
2. **Navigate to the project directory**:
   ```bash
   cd Weather-Widget
   ```
3. **Install the dependencies**:
   ```bash
   npm install
   ```
4. **Environment Variables**:
   If the project requires an API key (e.g., OpenWeatherMap), there is an existing `.env` file that you may need to configure with your keys.
5. **Run the development server**:
   ```bash
   npm run dev
   ```
6. **Open in Browser**:
   The app will typically be available at `http://localhost:5173`.
